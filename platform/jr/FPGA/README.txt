The .sof files are expected FPGA loads for the C256jr hardware:
CFP9599-4A.sof  - for the unmodified, first-run dev hardware.
CFP9599-16A.sof - for the updated first-run dev hardware.

The .asm files contain the 64tass definitions of the registers
provided by the above FPGA loads.
